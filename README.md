# clippy
Create *animated ASCII-art* for the command line!

clippy is your *friend* if you are;
- Developing a rogue-like ASCII game
- Creating a TUI
- Wanting to make ASCII-art
- Wanting to make ***animated*** ASCII art

Example: Ex directory art/sea contains "frames" (ASCII art txt file) 
of a boat at sea. Each file represents a frame of the final animation.
{file 1 + file 2 + ..... file n} becomes {gif}
